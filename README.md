## What is AutoJs?
AutoJs is javascript running without a browser and can use Autoit Features and other


### Exmple Code
``` JavaScript
try {
    MsgBox(0,"AutoJs","Hello AutoJs")
} catch (error) {
    ConsoleWrite("Error : " + error.message)
}

Exit()
```
### Run Code
	AutoJs index.js
	
## Features
- Threads
- Objects
